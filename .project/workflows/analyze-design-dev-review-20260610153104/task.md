# Task

Aligner le rendu affiché des CV et lettres de motivation avec le fichier PDF téléchargé.

## Source

Demande ad hoc utilisateur : « le rendu des cv et lettre de motivation ne match pas avec le fichier téléchargé, aligne le rendu au modèle de rendu ».

## Scope

- CV et lettre de motivation dans l’éditeur utilisateur.
- Le PDF téléchargé reste le modèle de référence.
- Le contenu généré, les routes d’export et le DOCX ne changent pas.
